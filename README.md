# 加工時間シミュレーター (Codex対応)

## 概要
本プロジェクトは、製造工程における加工スケジュールを最適化するAIシミュレーターです。
既存のHTML UIを維持したまま、裏側でCodexが制約条件を考慮したスケジューリングを行います。

## ディレクトリ構成
- `ui/` - 既存のHTML+JS UI（変更禁止）
- `src/scheduler.py` - Codexが処理を実装する場所
- `prompt/` - Codex用プロンプト指示
- `data/` - 入出力JSON例
- `tests/` - ロジック検証用

## 使用方法
1. `src/scheduler.py` にCodexがロジックを記述
2. `index.html` をブラウザで開き、入力後「計算」ボタンを押下
3. 計算結果はCanvas上に描画される

## 開発方針
- UIは壊さない
- ロジックはCodexが主担当
- 再現性と可視性の高い構造

## 例データとテストの利用
`data/` ディレクトリと `tests/` ディレクトリを作成した後は、以下の手順で
サンプルデータの読み込みやテスト実行が行えます。

### 例データの読み込み
1. `data/` 配下に任意の JSON ファイル（例: `example.json`）を配置します。
2. UI 画面で **読み込み** ボタンを押し、先ほどの JSON ファイルを選択する
   と、入力内容が自動で反映されます。

### テストの実行
1. `tests/` 配下にテストコードを配置します。 `pytest` を利用する想定です。
2. 依存関係をインストール後、リポジトリのルートから次のコマンドを実行して
   テストを走らせます。

```bash
pip install -r requirements.txt  # 必要に応じて
pytest
```

テストが成功すると、実装したスケジューリングロジックの検証結果を確認でき
ます。
